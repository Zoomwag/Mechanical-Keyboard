## Overview

This is a custom mechanical keyboard that I designed from scratch. It is a **60% layout** and runs on a **Raspberry Pi Pico** using **QMK firmware**.

The goal of this project was to make the keyboard:
- Easy to build  
- Affordable and accessible  
- Fully DIY-friendly  

All components are **through-hole (THT)**, making soldering much easier—especially for beginners. The entire keyboard is assembled using just **four M3 screws**, keeping the construction simple and clean.

The case is fully **3D printable**, and the keycaps can either be 3D printed or purchased. I recommend buying keycaps for better durability and finish.

The keyboard uses **Cherry MX-style switches**, so you can choose whichever type suits your preference. I will likely be using **Cherry MX Red switches** for a smooth, linear feel.
